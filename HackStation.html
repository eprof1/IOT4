Raspberry Pi Buster, Setup and Magic Mirror

FormativePi01 Start

Follow these instructions to install the Raspbian Buster operating system, and the Magic Mirror software onto a Raspberry Pi 3 B+ (does MagicMirror work on Buster yet?)

1. Go to https://www.raspberrypi.org/products/raspberry-pi-3-model-b-plus/ and review the latest instructions.

2. Go to https://projects.raspberrypi.org/en/projects/raspberry-pi-setting-up/2 and review the instructions for part you will need:

a. Raspberry Pi 3 B+

b. Power Supply (5V, 2.4A)

c. Micro SD card and adapter (16 or 32GB)

d. Keyboard and mouse

e. HDMI cable

f.  Network cable

3. Download the full version of Raspbian Buster with Desktop (1.1GB zip file), from: https://www.raspberrypi.org/downloads/raspbian/

4. Download Etcher, a tool to setup your Micro SD card, from: https://www.balena.io/etcher/

5. Install the Raspbian Operating System on your Micro SD card (may need adapter) using Etcher.

6. Insert your Micro SD card

7. Per instructions at:  https://projects.raspberrypi.org/en/projects/raspberry-pi-setting-up/4  Connect your:

a. Keyboard

b. Mouse

c. Network cable

d. HDMI cable

8. Plug in the Power Supply and startup your Pi.
ref: https://projects.raspberrypi.org/en/projects/raspberry-pi-setting-up/5

2?.   Install Magic Mirror software by opening a terminal window and executing the command:

bash -c "$(curl -sL https://raw.githubusercontent.com/MichMich/MagicMirror/master/installers/raspberry.sh)"

11. Configure the Magic Mirror-set it to Auto Start by adding startup script found at the "Make a MagicMirror start script." section found at:

https://github.com/MichMich/MagicMirror/wiki/Auto-Starting-MagicMirror

12. Configure the MagicMirror to use OpenGL and rotate the screen (optional)
ref: https://github.com/MichMich/MagicMirror/wiki/Configuring-the-Raspberry-Pi

13. Connect to the Raspberry Pi using Putty
ref: https://www.putty.org/

14. Connect to the Raspberry Pi using VNC Viewer
ref: https://www.realvnc.com/en/connect/download/viewer/

15. Update or change the compliments module
ref: https://github.com/MichMich/MagicMirror/tree/develop/modules/default/compliments

16. Rotate the Raspberry Pi screen from landscape to portrait
sudo nano /boot/config.txt
display_hdmi_rotate=1
sudo reboot
Note that the value of display_hdmi_rotate will be either 1 or 3 for portrait mode.

17. Add the Arduino IDE to the Raspberry Pi
sudo apt-get install arduino
The Arduino IDE will now appear under Start>Programming>Arduino

18. Add the mu (micro Python) editor to the Raspberry Pi for programming the micro:bit
sudo apt-get install mu -y
