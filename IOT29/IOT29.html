Python GUI on Raspberry Pi



400 Points
Learning Objectives

This assignment introduces ways to create Python GUI interfaces on a Raspberry Pi.
You can use either the BreezyPythonGUI by Lambert or use the PySimpleGUI

Assignment Overview

The purpose of this assignment is to gain experience creating Python GUI applications from existing Python code.
This shows you how to create simple user interfaces for others who will be using your work.
